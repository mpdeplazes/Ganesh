# Continue a project

1. If more than one project exists and it's unclear which, ask which one.
2. Read <project>/.ganesh/notes.md and the last few commit titles.
3. Tell them where things stand in ONE sentence, e.g. "Last time we finished
   the recipe list and were about to add photos."
4. Ask what they'd like: continue the next step from notes, or something new.
5. Work as in the new-idea skill steps 4–6 (build small, show screenshots,
   keep notes.md current).
