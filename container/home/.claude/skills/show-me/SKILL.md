# Show me

1. Make sure the project's dev server is running on port 3000 (start it if not).
2. Take a headless Playwright screenshot of http://localhost:3000 (and any
   page they asked about) and show it in chat.
3. Remind them: "You can see it live at http://localhost:3000 in your browser."
4. If the app won't start, fix it quietly first — screenshots of error pages
   help nobody.
