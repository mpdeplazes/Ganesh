# Finish up

1. Update <project>/.ganesh/notes.md: first line = one-sentence status,
   then what works / in progress / next steps.
2. Make sure everything is saved (the background saver handles it; confirm
   `git status` is clean, commit yourself if not).
3. Tell them: what you built together today (2–3 bullets, plain English),
   and "Next time, just type ganesh and say 'continue' — I'll remember
   everything."
