# Get help

1. Run `bin/ganesh help` — it creates a text file (usually on their Desktop).
2. Tell them exactly: "Text or email that file to your helper — the person
   who set this up for you — and tell them in one sentence what you were
   trying to do."
3. If they set this up themselves and have no helper, point them at the
   project's GitHub issues page instead.
4. Reassure them: nothing is lost; their projects are saved automatically.
